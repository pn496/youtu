アプリケーション開発
YouTube動画検索アプリ

１、Vue.jsとaxiosを使用してWebAPIを操作する
２、YouTube Data v3を使用する
３、Google Developers ConsoleにてAPIキーを入手
４、コード上にてaxiosでクエリパラメータを設定する
Ajax通信「Asynchronous JavaScript + XML」で動画検索アプリを作成。
※Webブラウザ上で動作するJavaScriptプログラムにより、Webサーバから非同期通信(Asynchronous)
でXMLデータを取得し、取得したデータをWebコンテンツに動的に反映する手法
〜Ajax通信ライブラリ〜
Promiseという非同期処理機能をベースとしたライブラリ『axios(アクオシス)』を使用する
スクリプトコード: 「https://unpkg.com/axios/dist/axios.min.js」
※「html」と「.js」の2つのファイル構成で実装する
